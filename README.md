# 🌍 LinguaVoice AI

An AI-powered multilingual voice chatbot designed to provide seamless communication across multiple languages using Google Gemini AI. LinguaVoice AI enables users to interact naturally through both voice and text while automatically detecting the user's language and responding intelligently.

---

# 🤖 AI-Powered Multilingual Voice Chatbot

LinguaVoice AI is an intelligent conversational assistant that combines Artificial Intelligence (AI), Natural Language Processing (NLP), Speech Recognition, and Text-to-Speech technologies to deliver real-time multilingual conversations.

The chatbot understands user queries, detects the spoken or typed language automatically, maintains conversation context, and responds naturally in the same language.

---

# 📖 Project Overview

Communication across different languages can be challenging. LinguaVoice AI eliminates language barriers by allowing users to speak or type in their preferred language while receiving intelligent responses in the same language.

The application is designed with a modern responsive interface and supports voice-based interaction, making conversations more natural and accessible.

# ⚙️ How It Works

1. User enters a message by voice or text.
2. The application detects the language automatically.
3. Voice input is converted into text.
4. The query is sent to Google Gemini AI.
5. Gemini generates an intelligent response.
6. The chatbot displays the response.
7. The response is spoken aloud using Text-to-Speech.
8. Conversation history is saved for future reference.
