# 🌍 LinguaVoice AI

An AI-powered multilingual voice chatbot built using Google Gemini API.

## 🚀 Features

- 🌐 Automatic Language Detection
- 🎤 Voice Input (Speech-to-Text)
- 🔊 Voice Output (Text-to-Speech)
- 🤖 Google Gemini AI Integration
- 💬 Intelligent Conversations
- 📁 File Upload Support
- 🖼️ Image Understanding
- 🌙 Dark & Light Mode
- 📱 Responsive Design

## 🛠️ Technologies Used

- React.js
- Vite
- Tailwind CSS
- Node.js
- Express.js
- Google Gemini API
- Web Speech API

## ▶️ Installation

```bash
npm install
npm run dev
```
